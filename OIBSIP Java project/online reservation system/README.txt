# 🎫 Java Reservation System

A simple console-based **Reservation Management System** built in Java, supporting user login, reservation creation, and cancellation.

---

## 📂 Project Structure

```
├── Main.java          # Entry point of the program
├── Login.java         # Handles user authentication
├── Reservation.java   # Manages reservation details
├── Cancellation.java  # Handles reservation cancellations
├── Database.java      # Stores and manages application data
└── README.md          # Project documentation
```

---

## 🚀 Features

- 🔐 **User Login System** — Secure authentication before accessing the system
- 📋 **Create Reservation** — Book and store reservation records
- ❌ **Cancel Reservation** — Remove existing reservations easily
- 🗄️ **Basic Data Management** — In-memory data handling via a dedicated Database class

---

## 🛠️ Technologies Used

- **Java** (JDK 8+)
- **Object-Oriented Programming (OOP)** principles

---

## ▶️ How to Run

1. Make sure **Java** is installed on your system:
   ```bash
   java -version
   ```

2. Navigate to the project folder:
   ```bash
   cd path/to/project
   ```

3. Compile all Java files:
   ```bash
   javac *.java
   ```

4. Run the program:
   ```bash
   java Main
   ```
